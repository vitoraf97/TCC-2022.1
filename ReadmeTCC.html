RESUMO
O presente TCC visa estudar e implementar as tcnicas de Inteligncia Artificial,
em especial de Deteco de Ataques e de Classificao Multi-Label, que esto
sendo utilizadas pela comunidade cientfica e indstria como parte da soluo
do problema de ataques de segurana de informao s redes de computadores
modernas e no desenvolvimento de Sistemas de Deteco de Intruso. O
trabalho utiliza dados gravados de redes reais abertos e disponveis na Internet,
e inclui uma descrio das caractersticas desses dados e do processo utilizado
para sua gerao. implementado um novo algoritmo para seleo de
caractersticas (Features) gravadas da rede de computadores de forma a
melhorar o tempo de execuo dessa seleo com baixa perda de qualidade.
implementado um algoritmo para reduo controlada do tamanho do arquivo
original de gravao para evitar tempos de processamento excessivos.
implementado um algoritmo para preparar o arquivo reduzido de gravao para
processamento Multi-Label. So implementados algoritmos para classificao
Multi-Label com indicao do tipo de ataque realizado, utilizando tcnicas de
Transformao do Problema e Adaptadas. O trabalho utiliza a linguagem Python
e suas bibliotecas SkLearn e SkMultiLearn. Os resultados obtidos so avaliados
atravs de mtricas de desempenho.
ABSTRACT
This TCC aims to study and implement Artificial Intelligence techniques, in
particular Attacks Detection and Multi-Label Classification, which are being used
by the scientific community and industry as part of the solution to the problem of
information security attacks on modern computer networks and in the
development of Intrusion Detection Systems. The paper uses data recorded from
real networks that are open and available on the Internet, and includes a
description of the characteristics of this data and the process used to generate it.
A new algorithm for selection of recorded computer network characteristics
(Features) is implemented in order to improve the execution time of this selection
with low loss of quality. An algorithm is implemented for controlled reduction of
the original recording file size to avoid excessive processing times. An algorithm
is implemented for preparing the reduced file for Multi-Label processing.
Algorithms are implemented for Multi-Label classification with indication of the
type of attack performed, using Problem Transformation and Adapted
techniques. The work uses the Python language and its libraries SkLearn and
SkMultiLearn. The results obtained are evaluated using performance metrics.
